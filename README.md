# hiyontof

HTML/JavaScript based DodontoF client for smart devices.

HTML/JS で動くスマートフォン向けどどんとふクライアントです。

## 設置手順

以下2つのコマンドをリポジトリ内で実行した後、
main ディレクトリの中身をまとめて任意のディレクトリにデプロイしてください。

``$ git submodule init``   
``$ git submodule update``
